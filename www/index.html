About This Demo

This map displays San Francisco transit data over a background map created using the latest satellite imagery from Planet. FME reads transit data from the SFMTA website and mosaics the most up-to-date imagery available for this area from the Planet Explorer program. The map is generated as a tile cache for zoom levels from 10 to 16, and Leaflet (a JavaScript library) is used for displaying the map.
